## 0. 我拒绝做的「默认烂版本」（承诺：绝不做这个）

一个灰色方盒房间，里面摆三个一模一样的方块，标着“思考 / 物件 / 音乐”。平涂均匀光、没有阴影没有氛围，
循环播放一段低质氛围嗡鸣，没有天气，声音不空间化，相机在方块之间生硬瞬移。点开就是一个 3D 简历模板，
**换谁都一样**。这正是上一批的失败形态。**绝不做这个。**

## 1. 五个截然不同的内心世界概念

每个 = 一句“感觉” + 那个让人忘不掉的细节（钩子）。

### ★ A. 潮汐图书馆 / The Tide Library（**主推**）
- **感觉**：一个把记忆当作水来盛放的人——退潮般安静、念旧、辽阔却不孤独。一座暖灯的读书回廊，立在一片
  只有几厘米深、却完美倒映整片星空的镜面水上；潮水极缓地涨落，旧书在水面轻轻浮沉。
- **钩子**：**你走过的每一步都在星空的倒影上荡开涟漪，瞬间打碎又重聚那片星海**；水面之下沉着你写下的字，
  涨潮时浮起、可读。**“楼层”不是楼梯，而是潮汐**——水位涨落露出/没入不同回廊。
- **内在映射**：思考＝沉入水中的字与案头的信；物件＝浮在水上的发光陈列（一圈浮岛）；影音＝中央黑胶/随乐明灭的灯（音乐＝潮的节拍）。
- **配色**：午夜蓝 / 银 / 暖琥珀（室内灯）/ 极光青。冷的水与天 × 暖的灯与书 = 强对比。
- 真实参照见 `01_uyuni_milkyway_mirror.jpg`、`02_uyuni_boat_mirror.jpg`（天空之镜）+ `03/04` 暖库的灯与书。

### B. 雨夜阁楼 · 不灭的唱片 / Rainy Attic（温和演进，最贴作者气质）
- **感觉**：被雨困住、哪儿都不必去的安全感。一盏灯、一张转不停的黑胶、一整墙旧书。
- **钩子**：凑近天窗/窗上的雨痕，每一道流下的水痕其实是**一句没说出口的话**（hover→文字浮现）。雨＝未寄出的念头。
- 风险：最接近现有 build，需警惕“在旧的简陋版上抛光”，要重做而非打磨。

### C. 记忆灯塔 / The Memory Lighthouse（垂直 + 显隐机制）
- **感觉**：雪夜悬崖上的灯塔，一层层往上爬，每层是一段年代的记忆；塔顶的光扫过黑海。
- **钩子**：**旋转的塔光**——扫过室内时点亮黑暗中本看不见的字与物件，扫到窗外时瞬间照亮雾里漂浮的“记忆岛”。光＝注意力。
- 映射：楼层＝人生章节、塔光＝回想、雪＝遗忘。

### D. 苔庭 · 雾中山居 / Moss Court（最安静的一个，禅）
- **感觉**：破晓的雾、纸障、苔藓庭院、远处一声钟。
- **钩子**：穿过纸障的体积光与雾；凑近水墨卷轴，墨会**晕开/活过来**成写下的思考；石钵滴水，每滴是一次呼吸的环境音。
- 映射：卷轴＝思考、枯山水＝被安放的记忆、钟＝回归的主题。配色走绿/灰/木，与其余几个区分。

### E. 霓虹雨 · 赛博阁楼 / Neon Rain（当代 lo-fi）
- **感觉**：凌晨三点，雨城上方的小阁楼，lo-fi 节拍，霓虹透过湿窗。
- **钩子**：窗上的雨把霓虹折射成在暗室里游动的彩色 bokeh 光斑；一台 CRT/全息播着故障感 VHS 记忆；窗外城市是活的。
- 映射：屏幕＝影音、桌＝思考、城市＝被退守的外部世界。

## 3. 我的满分定义（一句话）

> **满分** = 进门 60 秒内安静下来、并在某一刻“倒吸一口气想截图”；十分钟后还不想走——因为这个空间像
> **某一个具体的人的内心在缓慢呼吸**，而不是一个换谁都行的 3D 模板。

## 4. 主推 A 对“完成标准四问”的预答（开干前的目标，不是结论）

1. **哪 3 个瞬间想截图**：① 第一步踏上水面、星海在脚下碎开；② 暖灯回廊整个倒映在镜水里、悬在星空正中；③ 一本书浮起、翻开成一束光，字浮出水面。
2. **哪一个元素别人想不到**：把“楼层/进度”做成**潮汐涨落**，且地面是**可被脚步打碎的星空倒影**——水既是导航、是时钟、也是情绪。
3. **头 60 秒情绪弧**：黑场只有水声渐起 → 一盏灯亮、照出脚下竟是星空 → 试走一步、涟漪炸开星海（“我站在天上”）→ 远处暖灯回廊浮现 → 潮声里第一段字浮出水面。从“悬浮/迷失”到“被一盏灯接住”。
4. **为何待 10 分钟**：潮汐循环不断改变可达区域与浮现内容；写下的字会沉入水、下一次涨潮再浮起来给你重读；安静、有节律、像一个会呼吸的房间。
