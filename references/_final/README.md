# 成品包 · 潮汐图书馆 / The Tide Library

> 一个人的内心世界，做成可走进去、可参观、可停留、并随他生长的三维空间。
> 完整取舍与自纠循环见 [../../DECISIONS.md](../../DECISIONS.md)；守则见 [../../CLAUDE.md](../../CLAUDE.md)。

## 跑起来
```bash
cd D:\desk\ds\lingjing
npm install   # 已装可跳过
npm run dev    # 浏览器打开，点「进入」开声场 + 锁定指针（建议戴耳机）
```
进入后：从高空俯冲落到水面 → WASD 走在星海上（每步荡开涟漪）→ 准心点书墙/浮岛/留声机走近 →
在书墙写下一段思考，看它化作一点暖光沉进水里、随潮汐浮起。

## 完成标准 · 四问（自评，最终由你判）
1. **3 个想截图的瞬间**：落定第一眼 / 低头的星海 + 暖思绪 / 写字沉水那一下。
2. **独一无二的钩子**：走在可被脚步打碎的星空倒影上；**写下的念头化作暖光沉进星海、随潮汐浮回**。
3. **头 60 秒情绪弧**：高空环绕俯瞰 → 俯冲落水 → 脚下涟漪炸开星海 → 眼平接住，暖灯回廊在四周亮起。
4. **为何待 10 分钟**：写思绪看它入水入海、潮汐让整片思绪星座明灭起伏、登观星台坐下听肖邦、逐个走近书墙/浮岛/留声机。
