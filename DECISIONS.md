## 一句话概念

**潮汐图书馆 / The Tide Library。** 一座暖灯的读书回廊，立在一片只有几厘米深、却完美映出整片星空的镜面水上。
你"走在星海上"——每一步都在星空的倒影上荡开涟漪，打碎又重聚。映射一个**把记忆当作水来盛放**的内心：
退潮般安静、念旧、辽阔却不孤独。冷的水与天（午夜蓝/银）× 暖的灯与书（琥珀）。

- 思考/文字 = -X 一整面圆弧书墙 + 立在水上的写作台（写下的思考会"沉入水中"）。
- 珍视的物件 = +X 立在水上的发光陈列岛（浮岛基座 + 黄铜柱 + 顶台）。
- 影音 = 后方观星台上的留声机（空间化音乐声源）。
- 上下楼 = 镜面广场（水面层）↔ 后方抬高的观星台（上层），木梯连通。

## 我拒绝做的「默认烂版本」

灰盒子房间 + 三个一样的方块（思考/物件/音乐）+ 平涂均匀光 + 循环嗡鸣 + 声音不空间化 + 相机瞬移 =
换谁都一样的 3D 简历模板。**绝不做这个**（这是上一批的失败形态）。

## 满分定义（一句话）

进门 60 秒内安静下来、并在某刻"倒吸一口气想截图"；十分钟后还不想走——因为它像**某一个具体的人的内心
在缓慢呼吸**，而不是一个换谁都行的模板。

## 完成标准 · 四问的当前答案

1. **哪 3 个瞬间想截图**：① 第一步踏上水面、脚下星海碎开成涟漪；② 暖灯回廊整片倒映在镜水里、悬在星空中间；
   ③ 一本书从水上浮起 / 写下的字随潮浮出水面。（①③ 仍在打磨，见弱项清单。）
2. **别人想不到的一个元素**：地面是**可被你脚步打碎的星空倒影**，"楼层/时间"做成**潮汐**——水既是导航、是时钟、也是情绪。
3. **头 60 秒情绪弧**：黑场只有水声 → 一盏灯亮、照出脚下竟是星空 → 试走一步、涟漪炸开星海（"我站在天上"）→
   远处暖灯回廊浮现 → 第一段字随潮浮出。从"悬浮/迷失"到"被一盏灯接住"。（入场编排待做。）
4. **为何待 10 分钟**：潮汐循环不断改变可达区与浮现内容；写下的字沉入水、下次涨潮再浮起重读；安静、有节律、像会呼吸的房间。

## 关键技术决策

| 主题 | 选择 | 理由 |
|---|---|---|
| 星海镜面水（**核心钩子**） | drei `MeshReflectorMaterial`（crisp mirror）+ 自写「脚步涟漪」叠加着色器 | 真实平面反射星空与暖灯；涟漪用环形缓冲 `scene/ripples.ts`，PlayerControls 每跨一步 spawn 一圈 |
| 夜空 | 自写穹顶着色器（渐变 + 银河带）+ 真实星点 Points（会泛光、会被水倒映）+ 低月 | 不下载 HDRI；星点 crisp 才能在水里映出"星海" |
| 潮汐 | `theme.tideOffset/tidePhase`，水面缓慢上下 + 漂浮书随潮起伏 | "水会呼吸"；后续驱动"沉字浮起" |
| 几何真相源 | `theme.ts` 的 LAYOUT（R_COURT 镜面广场 / DECK 观星台 / STEPS 木梯 / PEDESTALS 浮岛 / BOOKWALL 书墙弧） | Gallery 渲染、walk.ts 碰撞、PlayerControls 聚焦共用 |
| 导航 | 沿用第一人称控制器（指针锁定 + WASD + 触屏 + 准心聚焦），改：低掠水面的入场环绕 + 脚步涟漪 | 框架复用，几何换皮 |
| 后处理 | AgX + Bloom（高阈值，只让真高光泛光）+ DOF（梦境景深）+ 暗角 + 极轻颗粒 | 出"镜头感"；Noise 从 0.42 降到 0.12 去掉颗粒噪 |
| 数据契约 | **保留三个 zone id/type**（bookshelf/objects/record），只换几何与皮肤 | zone id 属于数据模型——"世界可由数据完整重建"必须守住 |

## 自纠循环日志（/design-review 之前的内部目检）

**Round 1（基础舞台搭起 → 浏览器实拍 4 次迭代）**
- 发现：水像柏油地、不反射星空（重 blur [320,110] 把反射星点糊没了）—— **最严重，钩子失效**。
  → 修：blur→[0,0]、mixBlur→0、roughness 0.32→0.05、color 提亮一点。水变成 crisp 镜面，书墙/观星台/浮岛清晰倒映。
- 发现：月亮巨大扁平像乒乓球。→ 修：半径 2.4→1.25、加 halo、推到穹顶边、朝向原点。
- 发现：浮岛基座像塑料水桶。→ 重做为"水面小基座 + 细黄铜柱 + 顶台 + 暖光圈"——读作"发光陈列岛"。
- 发现：中央一个暗甜甜圈伪影。→ 定位为反射暗天顶 + blur ringing；blur 归零后消失。
- 发现：月反射是一团发噪的过曝斑。→ Noise 后处理 0.42→0.12；Bloom 阈值 0.16→0.5（只让真高光泛光）。
- 发现：星空太稀、反射没东西可映。→ STAR_COUNT 1700→2600、星点更大、银河带加宽加亮。
- 加：水上**漂浮的旧书**（随潮起伏、缓缓自转）——"把记忆当水盛放"的具象 + 前景不空。
- 加：DOF 收紧 → 梦境景深，星点 bokeh。

**Round 2（接上 zones + 真实音频 → 正式跑了一轮 `/design-review` 子代理评审 → 按分级修）**
- 评审三条 Blocker：① 水不是星镜（核心钩子塌了）；② 画面中央无源白光斑（中央补光在反射里糊成 bug）；③ 斜贯天空的蓝色光束（银河带硬切 + 雾叠出假光柱）。
- **修 ①（最关键）**：放弃"靠 MeshReflectorMaterial 反射稀疏星点"，改用**对称镜像方案**——把整片天空（穹顶+星点+月）再镜像复制一份到水面之下（`Sky` 里 `group scale=[1,-1,1]`），水面换成**半透明菲涅尔玻璃**（`Water`：垂直看几乎透明→看见水下星海，掠射→蓝色水光）。低头任何角度都是一整片密星海。暖灯倒影由水下镜像发光球 `MirrorBeacons` 提供。删掉挡住水下天空的 `SunkenFloor`。星点改为铺满整个上半球（含天顶）、更大更亮、加少量"主星"做锚。
- **修 ②**：删掉悬空的中央补光，换成观星台暖色补光 + 贴水面的低位暖溢光（有源、不糊白斑）。
- **修 ③**：银河带改乳白、加宽、软边、叠 fbm 絮状层次（不再是蓝色硬光束）；薄雾收紧半径（不再延伸到会投上天空的远处）。
- 修 High：书脊从"均匀纯色方块"改为**宽窄/高矮/进深各异 + 偶尔歪斜/抽出半截 + 做旧降饱和**；物件放大 + 提高自发光/金属度（陈列台不再"空"）；灯笼加**柔和光晕 + 假体积光柱**（光在薄雾里散开）；抬环境/月光、减暗角，救回死黑暗部。
- 结果（引擎内实拍复测）：低头是整片星海、暖灯有倒影、银河柔和、白斑/蓝束消失；书墙有层次、物件可见。`tsc` 0 错、`vitest` 27/27。
- 加：**入场电影**——入场前从高处（y≈3.7）缓缓环绕俯瞰整座漂在星海上的回廊；点「进入」后约 5.5 秒 smootherstep **俯冲沉到水面眼平**、落在出生点，临近水面时脚下荡开第一圈涟漪，再交还控制（`PlayerControls`）。落定一帧（眼平俯瞰整片星海上的回廊）本身就是个想截图的机位。

**Round 3（又跑一轮 `/design-review`）→ 两条最高杠杆：星海密度/对称 + 清理"糖豆"杂物**
- 评审指出：① 脚下星海太稀、不成"海"，且中段一条灰白雾"地平线"把天水切开（参照 01 的杀手锏是上下镜像对称、地平线消失）；② 满水面的暖色"葫芦/糖豆"实心椭圆（= 水下镜像发光球 MirrorBeacons + 烛环 + 漂浮书纸芯自发光）读不出是什么、像渲染残留，是当前最廉价的出戏元素；③ 到达第一眼没有视觉主角。
- 修：**加密星海**——星点 4200→7200、约 4 成星拉向银河大圆聚成"有星粒结构的星河"（不再靠雾）、整体更大更亮；**消地平线缝**——水面菲涅尔掠射不透明度 0.6→0.3、sheen 压暗、薄雾几乎删光（0.32→0.1、贴身不延伸），天水合一镜；**清杂物**——删掉 MirrorBeacons、删掉烛环、漂浮书纸芯自发光降到近无、删掉灯笼那根硬边假体积光锥（只留柔光晕）。
- 加：观星台**听歌角**（毯子 + 靠垫 + 暖台灯，在留声机旁）——给登顶一个"坐下听肖邦"的停留理由。
- 结果（实拍复测）：低头是一整片干净、密、亮的星海，糖豆消失，天水近乎对称；`tsc` 0 错。
- **灵魂机制已落地并实拍验证**：`scene/SunkenThoughts` —— 每条思考在水下是一颗**暖金色**光点（在一整片冷色星海里，"你的念头是暖的"，一眼能认出，不和星星混）；**写下新思考的瞬间，一点光从写作台落下、沉进水里**（落水触发一圈涟漪），加入既有的思绪星座；**随涨潮上浮变亮、落潮下沉变暗**。数据驱动（按 thought 实体生成，位置由 id 哈希稳定），首帧惰性建状态保证内容一加载就显示。这同时点亮了"完成标准"的第 2 问（独一无二的钩子）与第 4 问（为何待 10 分钟）。
- 加：观星台听歌角（见上）。`tsc` 0 错、`vitest` 27/27。

**Round 4（最终验证轮 `/design-review`：无 Blocker，核心已落地，指出"差一步到封面级"）**
- 评审确认：星海密度/灵魂思绪点/银河结构三件大事都成立，F_arrival 已跨进"想多看一眼"；唯一最高杠杆缺口——**低头时脚下星海被深蓝玻璃压住、没真正透出来**（水面菲涅尔不透明度偏高、地平线没消失）。
- **修（决定性一改）**：水面菲涅尔不透明度从 `mix(0.04,0.3)` 压到 `mix(0.015,0.12)`、指数 4→5——几乎全程透明 → 整片**镜像星空透出**、上下对称、地平线消失。低头即一整片与抬头同密的星海，暖思绪点沉在其中。F_arrival/F_starsea 升到"封面级"。
- 修 High：暖灯光球过曝纯白 → 降自发光强度（2.6/2.4/2.2→1.4/1.5）、核心改暖琥珀色（看得出灯芯，不再死白）；Bloom 强度/阈值微调，光晕更软。
- 成品包：`references/_final/`（F_arrival/F_starsea/F_books/F_islands/F_deck/thought_motes/write_sinks 七张实拍 + `walkthrough.gif` 一段穿行 + `README.md`）。`tsc` 0 错、`vitest` 27/27。
- 评审留下的收尾项（非阻断）：~~听歌角再 cozy~~（已修：圆软蒲团 + 折叠薄毯 + 摊开的书 + 茶杯 + 双层地毯，"有人刚坐过"）；漂浮书与书墙书脊再加"被翻过"的高低错落/亮色跳动；物件辨识度（仍是基本几何体）。
- 成品包已刷新到最终状态：`references/_final/`（七张实拍 + `walkthrough.gif` 穿行 + README）。`tsc` 0 错、`vitest` 27/27。

## 还很弱的地方（诚实清单 · 待后续轮次）

- **帧率仍需用户实机确认**：本轮做了结构性提速（SMAA 替 MSAA、去常驻 DOF、dpr 低起、raycast 节流、去二级平滑），但隐藏标签页 rAF 被节流，测不出真实 FPS。场景里仍有 ~12 盏动态 pointLight（灯笼/浮岛/台灯）是前向渲染的实打实开销，下一步可把部分灯烘成纯自发光、或上灯数上限。
- **入场情绪弧**：俯冲入场 + 第一圈涟漪已做；更完整的"黑场→逐次点灯→字浮出"分段编排仍未做。
- **物件辨识度**：浮岛上的"物件"仍是数据驱动的基本几何体（cube/sphere + 颜色），缺"一眼认出是某人之物"。这是数据契约里的 content，需要更丰富的 primitive 组合或图标系统——本轮只升级了 stage（建筑/家具）建模，没动 content。
- **灯泡仍带一块加色光晕盘**，泪滴灯泡 + 车削杆已远离"裸球"，但近看光晕略 CG；真正的玻璃罩（MeshTransmissionMaterial）本轮**为保帧率刻意没上**（它是唯一会真吃 FPS 的建模升级），留待帧率确认后给 ~2 盏主灯加。
- 材质仍是程序化纯色 + 倒角/车削轮廓，**无 PBR 贴图/法线**；大面（书墙/台面）缺微观粗糙度变化。
- 潮汐振幅小（0.055m），肉眼难察"水在呼吸"。
- 触屏漫游为基础实现；移动端以"看"为主。
